Preparing Data

Import Data

We use the normalized miRNA count matrix from GEO and merge it with the demographic metadata. Categorical variables are set to factors and the PDAC case and control variable is set to binary for modeling.

Create Training and Holdout Sets

To split the data while respecting matched-pairs, we performed the split on the matched set IDs, ensuring each pair is not split between sets.

Cross-Validation Loop

We perform a k-fold cross-validation on the training set of paired samples, ensuring matched-samples are kept together in the same fold.

Within each fold: 1. We apply a fold-specific feature filter, currently class-based signal-to-noise ratio (SNR) on the normalized counts, to select the top miRNAs within the training fold. 2. We then fit each model using only the selected miRNAs from the normalized count matrix.

The SNR filtering is calculated as the absolute difference in means between cases and controls divided by the sum of standard deviations, with an additional t-test for association. The top N miRNAs are selected based on the highest SNR values, with ties broken by p-value.
